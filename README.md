# Statistics Workshop: fundamentals

This initiative is a joint effort between the [Viral Ecology and Omics](https://www.bio.uni-jena.de/en/2235/viral-ecology-and-omics) Group of the [Friedrich Schiller University Jena](https://www.uni-jena.de/en) and the [Instituto Federal Goiano](https://www.ifgoiano.edu.br/home/index.php/rio-verde.html) (IF Goiano), Campus Rio Verde.
